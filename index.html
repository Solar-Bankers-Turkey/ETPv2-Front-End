Content Of the Package

- One (1) 23 Wp poly-crystalline solar panel with DC/DC converter for USB output.

- One (1) EasyAcc Monster 20,000 mAh Power Bank (4A input 4.8A Smart Output)

- Two (2) micro USB/USB cables

- One (1) Raspberry Pi 3 Model B +

- One (1) Raspberry Pi 3 Case

- One (1) Raspberry Pi 5.1V 2.5A Power Supply (UK, EU, USA and Australia)

- One (1) 16 GB MicroSD card preloaded with the latest version of NOOBS

Installing the Solar Bankers Node

Solar Node Operating System

Please download the image file of the solar bankers operating system. After downloading you can easily write the image file to the SD card with blena etcher. https://www.balena.io/etcher/

Use an SD card reader and connect it to your computer.Click Flash from file button and select the solar node iso file. After selecting the target you can start the flash button.

After flashing you can plug the sd card into raspberry pi.

Hardware

Before installing the Solar Bankers Node, make sure that:

  • The Power Bank is fully charged. To charge the device follow the instructions available in the package.
  • Insert the MicroSD card into the Raspberry Pi and place the device into its case.
    • Connect the Raspberry Pi to Power Bank
    • Connect the Power Bank to the Solar Panel.

That's all!

You can see the wire connections on this schematic.

Configurations

After you finnish the power connection solar node will open the wifi hotspot to connect and access the configuration page. You can connect the solar node network with your smartphone or computer.
